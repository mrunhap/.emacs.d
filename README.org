** 个人 =Emacs= 配置(WIP)

#+begin_src bash
emacs --batch --eval "(require 'org)" --eval '(org-babel-tangle-file "~/.config/emacs/config.org")'
#+end_src

*** Enable tabnine

Add this to =custom.el=:

#+begin_src emacs-lisp
(add-to-list 'prog-mode-hook #'tabnine-mode)
#+end_src

*** 如何输入上一条命令？

因为 =C-p= 在 Emacs 中默认是向上移动所以没办法像终端一样。
使用 =M-p= 或者 =up= 即可。
